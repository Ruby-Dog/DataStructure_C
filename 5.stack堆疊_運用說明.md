# 中序運算式處理:  

需使用到2個stack  

演算法的處理邏輯:  
```
1.建立兩個stack(堆疊) 運算元stack 和 運算子stack 初始為空;

2.從左往又開始逐一讀取運算式;

3.若讀取的是運算元,push入運算元stack;

4.反之,讀到的是運算子:
    { 1.若讀到 ( 左括弧 ,push到運算子stack; (中序表示法,左右括號優先權最低)
      
      2.若讀到 ) 右括號 ,依序彈出運算子stack內的運算子,直到彈出 ( 為止 。 
      　每次彈出一個運算子,就從運算元stack內彈出 2個運算元,
        並計算之，然後將結果推入運算元堆疊內;
      
      3.若讀取到非以上之任何運算子,與運算子stack頂端元素比較,
         if (stack為空 | 優先權高於該頂段元素 ){
              就push到stack內;
         }else{
               依序從stack內彈出運算子,
               直到頂端元素的優先權低於該運算子,
               然後將其push到stack;
         }
         
         每次pop出一個運算子 , 就依據該運算子所需的運算元數量 (例如2個) ,
         從運算元stack內,pop出相同數量的運算元(例如2個),依據運算子計算後,
         結果推入運算元堆疊內;
     }   
         
5.當運算式讀完後,若運算子stack非空,依序彈出運算子,並計算後,結果push入運算元stack,
  直到運算子stack為空,計算結束;
  
6.從運算元stack彈出最頂端元素,就是計算結果;
```
