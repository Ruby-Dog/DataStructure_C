### 何謂優先佇列
```
* Priority Queue
```
```
* 是一種進階佇列。
```
```
* 一種有序串列，
  其資料離開佇列，非依據先進先出原則，
  而是依據佇列元素的優先權，
  優先權高的先出，若優先權相同，才依據先進先出原則。
```
```
* 可以使用陣列或鏈結串列來實現。
```
### 以陣列實現優先權佇列
```
使用一維陣列，其內部依據優先權分隔成數個區塊，
每個區塊相當於一個一般的佇列，
優先權高的一般佇列空時，才換次低優先權佇列送出資料。
```

### 小結
```
* 優先權佇列是依據優先權決定資料刪除(移出佇列)的順序，
  優先權高的先刪除。
```
```
* 在優先權佇列內，每筆資料都有搭配一個優先權。
```
```
* 優先權佇列可以使用陣列或鏈結串列來實現。
```
